# Dataset Description: COVID-19 Global Statistics (2026)

## Overview
This dataset contains global COVID-19 statistics as of February 16, 2026.

## Data Source
The data that are from Kaggle are stored in `covid19_global_statistics_2026.csv` and contains records for over 200 countries and territories, as well as continental aggregates.

## Column Definitions
- **continent**: The continent where the country is located.
- **country**: The name of the country or territory.
- **population**: The total population of the country.
- **date**: The date the record was updated (all records in this set are 2026-02-16).
- **new_cases**: Number of new cases reported on the current date.
- **active_cases**: Number of currently infected individuals.
- **cases_per_million**: Total cases per 1 million people (useful for comparing countries of different sizes).
- **total_cases**: The cumulative number of confirmed cases.
- **new_deaths**: Number of new deaths reported on the current date.
- **deaths_per_million**: Total deaths per 1 million people.
- **total_deaths**: The cumulative number of confirmed deaths.
- **tests_per_million**: Total COVID-19 tests conducted per 1 million people.
- **total_tests**: The cumulative number of tests performed.
